# ⚛️ React Playground (Vite + React)

Projeto criado com **Vite + React** com o objetivo de **aprender e praticar conceitos fundamentais do React** por meio da criação de componentes reutilizáveis e interativos.


👉 **Acesse o projeto online:**  
🔗 *https://luscap3.github.io/React-Playground/*


---

## 🎯 Objetivo do Projeto

- Praticar conceitos básicos e intermediários de React
- Criar componentes reutilizáveis
- Trabalhar com rooks (`useState`, `useEffect`e `useRef`)
- Entender melhor eventos e renderização condicional
- Usar Vite para um setup rápido e moderno

---

## 🧩 Componentes Desenvolvidos

### 🎚️ Slider
- Controle de valores numéricos
- Atualização dinâmica de estado

### 🔘 Switch Button
- Botão de alternância (on/off)
- Controle de estado binário

### 📺 Carrossel
- Exibição de imagens em sequência
- Ao clicar, um evento **click** é executado

### 💳 Card
- Exibição de imagens
- Movimentar o mouse a perspectiva do **Card** é alterada
- Movimentação nas laterais causam um brilho
